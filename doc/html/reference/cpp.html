struct migraphx_compile_options¶

Options to be passed when compiling.

Public Members

bool offload_copy¶

For targets with offloaded memory(such as the gpu), this will insert instructions during compilation to copy the input parameters to the offloaded memory and to copy the final result from the offloaded memory back to main memory.

bool fast_math¶

Optimize math functions to use faster approximate versions. There may be slight accuracy degredation when enabled.
